Software Development Kits

Beschleunigen Sie Ihre Integration mit unseren offiziellen SDKs. Wir unterstützen die gängigsten Sprachen und Frameworks, damit Sie sich auf Ihr Business konzentrieren können.

Node.js SDK

Perfekt für Backend-Integrationen und Serverless Functions.

npm install @bestelldirekt/sdk

Nicht das Richtige dabei?

Sollten Sie eine Sprache verwenden, für die wir noch kein offizielles SDK anbieten, können Sie jederzeit direkt gegen unsere REST-Schnittstelle programmieren.

Zur API Referenz Postman Collection 
curl -X POST "https://api.bestelldirekt.de/v2/orders" \
     -H "Authorization: Bearer YOUR_TOKEN" \
     -H "Content-Type: application/json" \
     -d '{
       "pickup_id": "ST_442",
       "priority": "express"
     }'
